Issue: The array you're trying to map over is undefined.

Fix: Add a default value or conditional check:

// Option 1: Default value
const items = data?.items || [];

// Option 2: Conditional rendering
{data?.items && data.items.map(item => ...)}

// Option 3: Loading state
